# Boolean Builder — Visual Style Guide v1.0

*Design Direction: “Playfully Technical”*

## 1. Design Principles

* Playful but not childish
* Friendly clarity
* Delight without distraction
* Technical precision
* Modern SaaS polish

## 2. Color System

### Core Colors

* `--bg-app`: #F8FAFC
* `--bg-card`: #FFFFFF
* `--border-light`: #E2E8F0
* `--text-primary`: #1A1A1A
* `--text-secondary`: #4A5568
* `--icon-muted`: #A0AEC0

### Accent Colors

* Primary gradient: #8A6DFF → #C8A5FF
* Secondary mint: #3EE0C2
* Pastels: #EFE7FF, #E7F0FF, #E3FAF5

### Semantic

* Success: #38D897
* Warning: #F6AD55
* Error: #E53E3E

### Confetti Palette

* #F8D34A, #FF8A7A, #56E4C7, #7AC6FF, #C998FF

## 3. Typography

* Primary: Inter
* Monospace: IBM Plex Mono / JetBrains Mono

### Sizes & Weights

* Page title: 20–24px / 600
* Bucket title: 16px / 500
* Body: 14–15px / 400
* Labels: 13px / 400
* Operators: 14px / 600 (uppercase)
* Boolean preview: 14px monospace

## 4. Spacing

* 4, 8, 12, 16, 24, 32, 48px scale

## 5. Radius & Shadows

* Buckets: 12–16px radius
* Pills: full pill (999px)
* Buttons: 999px
* Shadows: subtle SaaS shadows

## 6. Component Styles

### Buckets

* White cards, soft borders, slight hover shadow
* Editable title, toggle, drag handle

### Pills

* Pastel backgrounds
* Drag + reorder friendly
* Delete icon
* Slight hover lift

### Operators

* Segmented control
* Uppercase bold text

### Add Terms Input

* Rounded, soft border
* Accepts type & paste

### Boolean Preview

* White card
* Monospace text
* Pretty vs Minified
* Optional syntax tinting

### Copy Button

* Gradient pill
* Hover glow
* Confetti on success

## 7. Motion

* Easing: cubic-bezier(0.4, 0, 0.2, 1)
* Term add: 0.9 → 1.05 → 1 scale
* Drag: elevated shadow
* Boolean fade updates
* Confetti: 10–20 pieces, 600–900ms

## 8. Accessibility

* Minimum contrast 4.5:1
* Clear ON/OFF states
* Confetti aria-hidden

## Summary

Comprehensive design foundation for UI, styling, and component consistency.
